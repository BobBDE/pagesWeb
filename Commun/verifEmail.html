Attention

L'adresse email cpe est assez restrictif sur les emails reçus. Il est possible que l'email prennent entre 15 et 20 min pour arriver dans vos boîte mail... De plus l'email sera mis dans vos "pourriels". Pour voir l'email vous devez aller dans votre boîte mail puis dans l'onglet préférences. Ensuite dans la catégorie Message cliquez sur l'onglet "Configurer les options de pourriels". Enfin dans le deuxième la deuxième question "Que faire des pourriels après qu'ils aient été déclarés pourriel ou légitime ?" Selectionnez Déplacer dans la boîte de réception.
